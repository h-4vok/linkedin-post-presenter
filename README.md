# LinkedIn Post Presenter

Standalone SPA to upload and browse AI-enriched LinkedIn post dumps.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Usage

1. Start the app.
2. Upload `linkedin_dump_2026-03-31_enriched.json` or any JSON file that follows the same schema.
3. Use the sticky header to search across the full JSON payload or keep the interest-only filter enabled by default.
